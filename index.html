FOR APPLICATION DEVELOPERS

Who want to use the latest tech, but don't want to waste time figuring out how to get it running.

  • Import blueprints that encode how to operate your favorite systems.
  • Deploy them to you favorite cloud provider without having to absorb reams of documentation.

FOR SYSTEM BUILDERS

Who want to make their favorite project easy to deploy.

  • Use an intuitive API to write a blueprint that describes how to operate your application.
  • Publish the blueprint to npm so that others can easily get up and running.
